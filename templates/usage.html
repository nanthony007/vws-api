API Usage

This page will serve as a tutorial/documentation for the API usage. This will include the GUI and direct API access.

---

Main url: 127.0.1:5000/api/v1

GET methods

Teams

Use main url + '/teams' to return all teams alphabetically.

Wrestlers

Use main url + '/wrestlers' to return all wrestlers alphabetically.

Wrestler Stats

Use main url + '/wrestlerstats/<"string:wid">' to get average statistics for a wrestler. Use the WID found from all wrestlers request url formatted.
For example, Nick Florida 141 becomes nick-florida-141 when inserted into the url GET request as <"string:wid">.
