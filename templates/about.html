About

For information on API usage please see the API Usage page.
---
This site was developed for the purpose of providing wrestler statistcs to developers. Over time, methods will be added to the API for more thorough access to statistics.
An API was chosen to allow faster/simpler access to data without directly accessing the database which would require authorization and conection strings. On that note, DELETE, PUT, and POST methods will almost certainly be unavailable as the primary purpose of this API is to grant data access via GET methods. Over time these methods will be expanded upon to improve performance.
